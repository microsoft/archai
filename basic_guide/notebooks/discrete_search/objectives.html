Evaluating models#

Objective functions are the main tool used to evaluate architectures in given criteria (task performance, speed, size, etc.). Objectives are optimized by search algorithms.

There are two types of objective function abstractions:

  • Objectives archai.discrete_search.Objective

    • Must override Objective.higher_is_better (optimization direction)

    • Must implement Objective.evaluate(model, dataset, budget)

  • Asynchronous Objectives archai.discrete_search.AsyncObjective:

    • Must override AsyncObjective.higher_is_better (optimization direction)

    • Must implement AsyncObjective.send(model, dataset, budget) and AsyncObjective.fetch_all()

Objective functions may optionally use a dataset (e.g task accuracy) or not (e.g architecture latency). Objective functions may also receive a budget value from some search algorithms. This budget value is used by some algorithms to sinalize how much budget the model evaluation should spend.

Read more about them here

Custom objective example#

[9]:

from archai.discrete_search import Objective, DatasetProvider

[10]:

class NumberOfModules(Objective):
    ''' Class that measures the size of a model by the number of torch modules '''

    higher_is_better: bool = False  # Smaller models are better

    @overrides
    def evaluate(self, model: ArchaiModel, dataset: DatasetProvider,
                 budget: Optional[float] = None):
        return len(list(model.arch.modules()))

[11]:

m = ss.random_sample()

[12]:

my_objective = NumberOfModules()
my_objective.evaluate(m, None, None)

[12]:

67

Some useful objective functions#

  • RayParallelObjective - Wraps an existing objective and runs it using multiple Ray workers

  • EvaluationFunction - Wraps a function that takes a (model, dataset_provider, budget) triplet and creates an objective

Parallelizing NumberOfModules#

[13]:

from archai.discrete_search.objectives.ray import RayParallelObjective

[14]:

my_objective_parallel = RayParallelObjective(
    NumberOfModules(), timeout=10, num_cpus=1.0
)

my_objective_parallel is now an AsyncObjective object

[15]:

model_list = [ss.random_sample() for _ in range(10)]

for model in model_list:
    print(f'Dispatching job for {model.archid}')
    my_objective_parallel.send(model, dataset=None, budget=None)
