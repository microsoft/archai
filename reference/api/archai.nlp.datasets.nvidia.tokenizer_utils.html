Base Vocabulary#

Utilities for tokenization pipelines with huggingface/tokenizers.

class archai.nlp.datasets.nvidia.tokenizer_utils.vocab_base.VocabBase[source]#

A customizable tokenization pipeline for encoding and decoding text.

This is an abstract base class that defines the interface for implementing a vocabulary class that can be used for tokenization. Subclasses should override the abstract methods in this class to provide specific implementations.

encode_file(path: str, verbose: Optional[bool] = True) → torch.Tensor[source]#

Encode text from an input file.

This method reads text from the specified file and encodes it using the encode_text method. It also includes options for verbosity and efficiently handling large datasets by converting the encoded tokens to a torch.Tensor every 500k lines.

Parameters

  • path – The path to the input file.

  • verbose – Whether to add verbosity to the logger.

Returns

The encoded tokens.
