class archai.nlp.eval.harness.harness_task.HarnessTask(dataset_name: str, dataset_config_name: Optional[str] = None, dataset_dir: Optional[str] = None, dataset_split: Optional[Union[str, List[str]]] = None, dataset_cache: Optional[str] = None, dataset_samples: Optional[int] = - 1, random_seed: Optional[int] = 42, num_proc: Optional[int] = None, metric_name: Optional[str] = None, metric_config_name: Optional[str] = None)[source]#

Abstract base class for harness-based tasks.

A harness-based task is a task that involves evaluating a model’s performance on a particular task using a dataset, a metric, and possibly some pre-processing steps.

HarnessTask subclasses must override the _create_inputs, _create_label, create_sampling_calls and compute_results method.
